# IPv4 vs IPv6
## IPv4
- 일반적으로 사용하는 IP주소
- 32비트의 크기로 2^32(약 42억)개의 IP가 존재
## IPv6
- IPv4의 부족으로 새로운 IP주소 체계
- 128비트의 크기로 무수히 많은 IP가 존재
- 규칙
    - 앞의 0은 생략 가능
    - 여러개의 0000 은 :: 으로 대체 가능
