# Servlet
- Java 클래스로 특정 네트워크 요청(ex. HTTP Request)에 대해 응답하는 역할을 한다
- 또다른 Java Application인 'Servlet container' 에 의해서 실행되며 대표적인 Servlet Container 로는 Tomcat 이 있다
	- 요청을 받음 -> 서블릿 컨테이너에게 요청을 전달 -> 적절한 서블릿으로 요청을 전달
