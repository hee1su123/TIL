# 오버로딩 vs 오버라이딩

## 오버로딩(Overloading)
- 하나의 메서드 이름으로 여러 기능을 구현 가능
- 같은 이름의 메서드도 매개변수가 다르면 정의 가능
- 리턴값만 다르면 오버로딩이 성립되지 않음(Java 기준)

## 오버라이딩
- 부모 클래스로부터 상속받은 메서드를 재정의하는 것
    - 그대로 사용할 수 도 있지만, 상황에 따라 다른 기능으로 변경하여 사용
- 상속 받은 것을 재정의 하는 것이므로 리턴값, 함수명, 매개변수 전부 같아야한다
