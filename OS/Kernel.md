# Kernel(커널)
![Kernel](../Image/OS/Kernel.png)
- 운영체제의 핵심 프로그램으로, 시스템의 모든 것을 통제
- CPU 에 명령을 내리는 소프트웨어
- 운영체제에의 핵심부로 컴퓨터 자원 등을 관리하는 역할
    - 컴퓨터의 자원을 바라보기 때문에 사용자와 상호작용을 지원하지 않음
    - 시스템 프로그램 Shell(시스템콜의 일종)과 같은 소프트웨어를 통해 사용자와 커널이 상호작용 가능
- 기능
    - 메모리 관리
    - 프로세스 관리
    - 장치 드라이버
    - 시스템 호출 및 보안
