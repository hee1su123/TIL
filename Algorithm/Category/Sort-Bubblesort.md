# Bubblesort(버블 정렬)
- 가장 쉽고 직관적인 정렬 방법
- 첫번째 두번째, 두번쨰 세번째, 세번째 네번째 ..... 마지막까지 비교하여 교환

## 버블정렬 이해하기
```
배열: [4, 6, 2, 9, 1]

1단계 : [4, 6, 2, 9, 1] 
        4와 6을 비교

2단계 : [4, 6, 2, 9, 1]
        6과 2를 비교 후 교환
        [4, 2, 6, 9, 1]

3단계 : [4, 2, 6, 9, 1]
        6과 9를 비교

4단계 : [4, 2, 6, 9, 1]
        9와 1를 비교 후 교환
        [4, 2, 6, 1, 9]

큰 수가 나오면 둘의 자리를 변경했으므로 가장 맨 뒤에 가장 큰 수가 있음. 맨 뒷자리를 제외하고 같은 과정 반복

5단계 : [4, 2, 6, 1, 9]
        4와 2을 비교 후 교환
        [2, 4, 6, 1, 9]

6단계 : [2, 4, 6, 1, 9]
        4와 6을 비교

7단계 : [2, 4, 6, 1, 9]
        6와 1을 비교 후 교환
        [2, 4, 1, 6, 9]

두 번째로 큰 숫자인 6이 맨 뒤에서 두번째에 위치. 맨 뒤 두자리를 제외하고 같은 과정 반복

8단계 : [2, 4, 1, 6, 9]
        2와 4을 비교

9단계 : [2, 4, 1, 6, 9]
        4와 1을 비교 후 교환
        [2, 1, 4, 6, 9]

세 번쨰로 큰 숫자 4가 맨 뒤에서 세번째에 위치. 맨 뒤 세자리를 제외하고 같은 과정 반복

10단계 : [2, 1, 4, 6, 9]
         2와 1을 비교 후 교환
         [1, 2, 4, 6, 9]

---정렬 완료---
```
